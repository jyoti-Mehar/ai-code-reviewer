# AI-Code-Reviewer
AI-powered full stack code review platform that analyzes JavaScript files, provide quality scores, security insights, line-by-line suggestions, and code previews using Node.js, Express, and frontend JavaScript.

## Tech Stack
- Node.js
- Express.js
- JavaScript

## Folder Structure

backend/
├── src/
│   ├── controllers/
│   │   └── analyzeController.js
│   ├── routes/
│   │   └── analyzeRoutes.js
│   ├── services/
│   │   └── codeAnalyzer.js
│   └── frontend/
│       ├── index.html
│       ├── script.js
│       └── style.css
├── app.js
├── package.json
└── package-lock.json

## How to Run Backend

1. Open terminal  
2. Navigate to backend folder  
3. Run npm install  
4. Run node app.js
